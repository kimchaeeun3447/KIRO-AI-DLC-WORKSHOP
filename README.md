# StarGazer

> 데이터야놀자 2026 · AI-DLC Workshop · 별자리 관측 플래너

> **"오늘 밤 우리 동네에서 어떤 별자리가 보일까?"**
> 위치/시간/광공해 조건을 입력하면 그 시점에 실제로 보이는 별과 별자리를 시각화하고 추천하는 웹 애플리케이션.



## 📋 개요

> ## ✨ 데이터로 별자리를 그리다: AI-DLC로 만드는 풀스택 서비스
>
> **데이터를 다루는 사람이라면 누구나, 자신의 데이터로 완성도 있는 앱을 직접 완성할 수 있습니다.**
>
> 본 워크샵은 AWS의 AI 코딩 도구 **Kiro**와 **AI-DLC(AI-Driven Development Lifecycle)** 방법론을 활용해,
> 항성 데이터로 *"오늘 밤 우리 동네에서 보이는 별자리"* 풀스택 웹 앱을 처음부터 완성하는 실습 과정입니다.
>
> 명확한 명세(Spec)와 의사결정이 있다면, **AI-DLC가 전체 개발 흐름을 안내하고, Kiro가 구현을 담당**합니다.
> 데이터에서 출발해 동작하는 서비스를 만드는 전 과정을, 처음부터 끝까지 직접 손으로 경험해 보세요!


## 🎯 워크샵 목표

- AI-DLC(AI-Driven Development Lifecycle) 워크플로우를 직접 경험
- Kiro와 함께 **요구사항 정의 → User Story → Application Design → 코드 생성**까지 전체 흐름 체험
- 데이터셋을 사용하여 **풀스택 웹앱**을 직접 완성하는 자신감 획득
- Vibe Coding 대비 AI-DLC가 만드는 **결과물의 일관성/정확성/확장성** 차이 체감
